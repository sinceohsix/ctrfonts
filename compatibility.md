## Custom Font Compatibility
Not all 3DS software handles custom fonts properly. This page documents any known compatibility issues and working titles.

If a game or app isnt listed here, it likely doesn't use the system font and should work. However, this list does not cover every piece of software made for the 3DS. If an app has any compatibility issues with custom system fonts, open an issue.

>[!NOTE]
>Custom fonts with characters that are generally bigger than the ones in the default 3DS font may have some spacing issues depending on the game and font used.

### Categories

[**Expected**](#expected) • System titles that use the system font and work with custom fonts as expected

[**Incompatible**](#incompatible) • Titles that have graphical or technical issues relating to installed fonts

[**Compatible**](#compatible) • Non-system titles that use the system font but have no issues with custom fonts

---

### Expected

- HOME Menu
- System Settings
- Nintendo 3DS Camera
- Nintendo 3DS Sound
- Activity Log
- StreetPass Mii Plaza
- Mii Maker
- FBI
- hShop
- the homebrew launcher
- amiibo Settings
- System Applets
  - Keyboard
  - Game Notes
  - Friend List
  - Notifications
  - Internet Browser
  - Miiverse

### Incompatible
Titles that have graphical or technical issues related to installed custom fonts

- None documented.
